ex. 음수 간선이 포함된 상황에서의 최단 거리 문제

양수 간선인 경우 다익스트라 사용 -> 음수 간선이 포함된다면?
-> 음수 순환이 발생 하여 최소비용이 -무한대가 될 수 있음

음수 간선이 있는 경우 + 음수 간선 순환이 있는 경우 = 벨만 포드 알고리즘
O(VE)의 시간복잡도(매번 모든 간선을 전부 확인하므로 다익스트라보다 느림)

1. 출발 노드 결정
2. 최단 거리 테이블을 초기화
3-1. 전체 간선 E개를 하나씩 확인
3-2. 각 간선을 거쳐 다른 노드로 가는 비용을 계산하여 최단 거리 테이블을 갱신
4. 음수 간선 순환이 발생하는지 체크하고 싶다면 3번의 과정을 한번 더 수행
-> 최단 거리 테이블이 갱신된다면 음수 간선 순환 존재
