1. 선택 정렬
처리되지 않은 데이터 중 가장 작은 데이터를 선택해 맨 앞에 있는 데이터와 변경
마지막 과정 처리 x
O(N^2)

2. 삽입 정렬
처리되지 않은 데이터를 하나씩 골라 적절한 위치에 삽입
앞쪽 데이터가 정렬되어 있다 가정하고 적절한 위치에 데이터를 삽입
O(N^2) (최선의 경우 O(N))
-> 거의 정렬되어 있다면 매우 빠르게 동작

3. 퀵 정렬
기준 데이터를 설정하고 그 기준보다 큰 데이터와 작은 데이터의 위치를 바꾸는 방법
첫번째 데이터를 기준 데이터(pivot)로 설정
O(NlogN) (최악의 경우 O(N^2))

4. 계수 정렬
데이터의 크기 범위가 제한되어 정수 형태로 표현할 수 있을 때 가능(매우 빠르게 동작)
O(N+K)
-> 동일한 값을 가지는 데이터가 여러 개 등장할 때 효과적으로 사용
 

표준 정렬 라이브러리는 O(NlogN)
